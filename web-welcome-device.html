Implementation Logic

Cara ini berfungsi (technical)

1
Detect viewport

React `useEffect` cek `window.innerWidth` pada mount. Re-cek bila resize. Cache dalam state.

if (w < 1024) showMobileBlock();
else showDesktop();
2
Render conditional

SSR-safe — render kosong dulu, masa client mount baru render mengikut viewport. Avoid hydration mismatch.

3
Optional escape hatch

"Teruskan ke web (terhad)" link untuk user keras kepala. Boleh tambah `?force=desktop` query param.

Rumitnya?

TIDAK rumit. ~30 baris kod dalam Next.js. 1 component `<DeviceGate>` yang wrap semua page. Implement dalam ~2 jam termasuk testing. Boleh tambah analytics tracking (berapa user kena block, conversion ke app download).
